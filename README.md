# 21 Game

## Project Description
This is a Python project for the classic "21 Game".  
The goal of the game is to reach a total of 21 or get as close as possible without exceeding it.

## How to Run
1. Make sure Python is installed on your computer.
2. Download the file `project_52.py` from this repository.
3. Open Terminal (Command Prompt) in the same folder as the file.
4. Run the following command:

![Game Screenshot](https://github.com/Mks-code1/python-21-game/raw/main/Screenshot.png
)


## How to Play
- When you run the game, you will be asked to make a choice.  
- To start the game, you **must select "Twenty One"**.  
- Follow the instructions in the terminal to play the game.  
- Try to reach 21 without exceeding it!

## Notes
- The game runs in the terminal.  
- You can modify the code to change game rules or add new features.
